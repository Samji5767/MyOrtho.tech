# Printing Workflow

## Workflow
1. Select treatment stage range and model count.
2. Choose layer height preset.
3. Adjust model orientation for surface quality and support risk.
4. Review resin and print time estimates.
5. Inspect printer fleet telemetry and material volume.
6. Refresh printer telemetry or queue jobs for manufacturing.
7. Export manufacturing report for lab and case documentation.

## Implemented Workspace
- Print preparation panel with model count, orientation, and layer height controls.
- Resin estimation and print time estimation.
- Support risk recommendation based on orientation.
- Fleet status cards with material levels and telemetry refresh.
- Queue table with case, stage, status, and QC score.
- Manufacturing, case, and export workflow descriptions.
